## 교체 후 6번 섹션 — "5월 로드맵 (하드닝 스프린트)"

내부 시드 30~50명으로 모든 핵심 모듈 실사용 검증. 외부 마케팅은 보류.

1) **마음트랙 30일 완주 검증**
   - 시드 10명 D1~D30 실주행, UTC 일수/회고 자동오픈/체크인 저장 엣지 픽스
   - 실패 리트라이 UI · 누락 미션 자동복구 점검

2) **워크북 모듈 다지기**
   - DailyMissionPreview / WorkbookPreviewCard / MilestoneProgressBar 모바일 회귀 QA
   - `mindTrackDayCopy` · `mindTrackDayKeywords` 톤 일관성 패스

3) **검사 시스템 신뢰도**
   - `/assessment?test=...` 단일 진입 잔존 라우팅 정리
   - 점수 저장(`mind_track_checkins.meta`) 일관성 테스트
   - 결과 → 워크북 회고 자동오픈 1회성 가드 회귀 테스트

4) **리포트 다지기**
   - Pro 리포트 출력 sanitization, PDF 분기 / 낙오 페이지 정리
   - 검증 QR(`/verify-report`) E2E

5) **게임상담 모듈 정비 + 신규 추가(미정 — 회의 안건)**
   - Geumjjok 허브 · VAD/TTS 안정화, 결과 PNG 내보내기 회귀
   - 신규 게임 모듈 1~2종은 회의에서 결정. **노출 정책은 별도 안건**으로 둠

6) **AI Copilot 다지기**
   - `/ai-copilot` 진입 동선·CopilotBubble 자동오픈 회귀
   - `copilotFlows` 분기 케이스 보강

7) **인프라/관측**
   - D1/D7/D14/D30 리텐션 측정 인프라
   - 결제 퍼널 이벤트 표준화
   - 위기 라우팅(`/expert-hiring?urgent=true`) 점검

**5월 내부 KPI**
- 시드 30~50명 확보
- D7 리텐션 ≥ 40%
- 결제 후 워크북 D3 진입 ≥ 70%
- 크리티컬 버그 0건 1주 유지

**회의 결정 필요 안건**
- 5월 신규 게임상담 모듈: 어떤 컨셉을 추가할지
- 신규 모듈 노출 정책: 내부 베타(시드 한정) vs 전체 공개

## 교체 후 7번 섹션 — "6월 GA 정식 론칭"

1) **마음트랙 ₩19,900 단일상품 정식 마케팅 ON** (가격 표기는 코드 기준)
2) **전문가 상담 매칭 v2** — `match-consultation-expert` 가동
3) **B2B Job Coach 파일럿 1곳 클로징** → 사례 공개
4) **영어버전 100% 완성본 동시 공개**
5) **5월 다지기 게임상담 모듈 GA 토글 ON**

**6월 KPI 목표**
- 결제 가입자 150명
- 결제 전환율(랜딩→결제) 3%
- D7 리텐션 40%
- 상담 1건 이상 사용자 비율: 결제자의 20%

**6월 GA 직전 체크리스트(요약)**
- 결제/환불/Billing Key 정상
- 핵심 라우트 404 0건
- Pro 리포트·PDF·QR 검증 정상
- 위기 라우팅 정상
- 영어버전 한글 잔존 0건
