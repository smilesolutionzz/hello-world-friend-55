

## 메인 페이지 "실제 학부모 반응" 섹션 이해도 개선

### 문제
- 현재 `RealFeedbackSection`은 카카오톡 대화 목업 + 거울식 하이라이트 카드로 구성되어 있는데, 처음 보는 사용자가 "이게 뭐를 보여주는 섹션인지" 직관적으로 이해하지 못합니다.
- "공포 → 이해 → 실천" 한 줄과 ₩19,900 CTA만 있을 뿐, 우리가 지금까지 만들어온 서비스(즉석 AI 분석, 30일 마음 트랙, 검사 20+, 전문가 매칭, 위험 감지 케어콜 등) 가치가 이 섹션에 응축되어 보이지 않습니다.

### 개선 방향

**1. 섹션 헤더 재작성 (의도 명확화)**
- 상단 칩: "이렇게 작동해요" / "30초 요약"
- 메인 카피: 
  - "**아이의 행동 한 마디**가 → **AI 분석 한 줄**로 → **30일 변화 트랙**이 됩니다"
- 보조 카피: "AIHPRO가 어떻게 도와드리는지 3단계로 보여드릴게요"

**2. 채팅 목업 위에 "1단계: 부모님 한 마디 → AIHPRO 해석" 라벨 추가**
- 채팅 박스 상단에 "STEP 1 — 부모님 말 → AI 전문 해석" 배지
- 메시지 수 9개 → 4개로 축소(가장 임팩트 있는 2쌍만): 짧고 빠르게 핵심만 전달

**3. 우측 패널을 "3단계 여정 카드"로 재구성**
현재의 before/after 3개 카드를 → 우리가 만든 서비스 흐름과 직접 매핑되는 3개 단계 카드로 교체:

```text
STEP 1  즉석 AI 분석
        한 마디 행동 묘사 → 30초 안에 전문 리포트
        (₩0 무료 체험)

STEP 2  30일 마음 변화 트랙
        매일 체크인 + 7/14/21/30일 전문가 개입
        위험 감지 시 무료 케어콜 자동 알림
        (₩19,900)

STEP 3  필요할 때 전문가 매칭
        '어려워요' 한 번에 1:1 화상 상담 연결
        (Day별 ₩9,900~₩99,000)
```
각 카드에 핵심 픽토(✅ 체크, 📈 추이, 💬 상담), 가격, "자세히 보기" 링크 포함.

**4. 하단 통합 CTA 재배치**
- 기존: 단일 ₩19,900 버튼
- 변경: 2개 CTA 병행
  - 메인: "30일 마음 변화 트랙 시작 — ₩19,900" (Primary)
  - 서브: "먼저 무료로 분석 받아보기" → `/instant-analysis` (Outline)

**5. 미니 서비스 라인업 띠 추가 (섹션 최하단)**
이 섹션 마지막에 "이외에도 운영 중인 기능" 한 줄 가로 스크롤 배지: 즉석 AI 분석 · 20+ 심리검사 · AI 음성 상담 · 전문가 매칭 · 저널/리포트 · 위험 감지 케어콜 · 종단 대시보드 · AI 코파일럿
→ 이미 만들어둔 `PlatformFeaturesShowcase`와 시각적으로 연결되어 "이 회사가 이만큼 만들었구나"가 한눈에 들어오게 함.

### 수정 파일
- `src/components/landing/RealFeedbackSection.tsx`: 헤더 카피·채팅 메시지 축소·우측 패널 3단계 여정 카드·CTA 이중화·하단 미니 라인업 추가
- (선택) `src/pages/Index.tsx`: `RealFeedbackSection` 위/아래 `LazyLoad minHeight` 값을 새 섹션 높이에 맞춰 조정해 스크롤 점프 방지

### 모바일 고려
- 우측 3단계 카드는 모바일에서 채팅 목업 아래로 자연 스택
- `break-keep` 적용, 카피는 12~14자 단위로 줄바꿈 제어
- 채팅 메시지 4개로 축소되어 모바일에서도 한눈에 들어옴
