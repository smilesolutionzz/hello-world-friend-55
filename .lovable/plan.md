`/b2b-center/app/admin/therapists` 페이지에서 치료사 색상을 자동으로 서로 다르게 배정해 일정표에서도 색이 구분되도록 수정하겠습니다.

01. 페이지 진입 시 색상 자동 배정
- 치료사를 불러올 때 `calendar_color`가 비어 있거나 다른 치료사와 중복인 경우, 일정표와 동일한 12색 팔레트에서 아직 쓰이지 않은 색을 자동 배정합니다.
- 배정된 색은 `center_therapists.calendar_color`에 백그라운드로 저장되어 일정표·캘린더에서도 즉시 같은 색으로 보입니다.

02. 색상 직접 변경
- 표의 색상 칸을 클릭하면 컬러 피커가 열려 색을 바꿀 수 있고, 변경 즉시 DB에 저장됩니다.

03. 한 번에 재배정 버튼
- 상단에 "색상 자동 재배정" 버튼을 추가해, 모든 치료사 색상을 팔레트 순서대로 깔끔하게 다시 칠할 수 있게 합니다.

기술 메모: `src/pages/b2b-center/console/TherapistsAdminPage.tsx` 한 파일만 수정합니다. DB 스키마/권한 변경은 없습니다(이미 존재하는 `calendar_color` 컬럼 사용).
