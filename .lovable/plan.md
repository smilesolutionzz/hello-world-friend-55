# 케어플 vs AIHPRO Center — 비교 & 이관 락인 전략

## 01 · 두 제품의 본질적 차이

### 케어플 (현재 시장 1위, 점유율 추정 70%+)
- **정체성:** "장애인 복지센터용 ERP/전자청구 시스템" (15년+ 업력)
- **강점**
  - 전자바우처 청구 (사회서비스원/국민연금공단) 직결 — 결제 인프라 락인
  - 회계·근태·계약·평가·교육 모듈 풀세트 (대형 센터 모든 워크플로우 흡수)
  - 보건복지부·지자체 보조금 신청 양식 자동 생성
  - 종이→디지털 전환 시대(2010s)에 자리 잡은 관성 (사용법을 직원이 외움)
- **약점 (= 우리의 침투점)**
  - UX: 윈도우 95급 테이블, 메뉴 50개+, 신규 직원 교육에만 2주
  - 모바일 사실상 미지원 (현장 치료사가 PC 앞에 와야 입력)
  - 데이터 분석/리포트: 엑셀 다운로드만, 시각화 0
  - 부모와의 소통 채널 없음 (B2B만, B2C 단절)
  - 가격: 월 15-25만원 + 셋업비, 약정 1년
  - AI 0%, 입력 노가다 100%

### AIHPRO Center (우리)
- **정체성:** "센터 운영 + 부모 코칭 + AI 인사이트가 하나로 묶인 차세대 플랫폼"
- **현재 보유 기능 (코드 기준)**
  - 운영 핵심 3: 이용자/일정/수납 (이수석 피드백대로 의도적 축약)
  - 케어플 엑셀 5종 직임포트 (`CarepleImportGuide` + `excelImport.ts`)
  - 전자바우처 청구 모듈 (`VoucherClaimsPage`, draft→approved 워크플로우, .xlsx 자동 생성)
  - 월말 마감 (`BillingStatsPage` + `center_billing_closings`)
  - 5단계 온보딩 위저드 (`/b2b-center/app/setup`)
  - 센터 스토어프론트 (`/center/:slug`) — 케어플엔 없는 B2C 수익 채널
  - 부모 리포트 (`ParentReportsPage`) — 부모와 직접 소통
  - 60일 무료 + 카드 등록 없음
- **약점**
  - 전자바우처 실제 청구 연동 (사회서비스원 API)은 .xlsx 생성까지만, 자동 전송 X
  - 회계/근태/세무 모듈 부재
  - 시장 인지도 0, 레퍼런스 센터 부족
  - 케어플에 익숙한 50대 원장에겐 "또 배워야 하나?" 저항

## 02 · 핵심 인사이트 — "왜 옮기지 않는가"

이관을 막는 진짜 장벽은 기능이 아님:

1. **데이터 이관 공포** — "15년치 이용자/회기/수납 데이터가 날아가면 어떡하지?"
2. **전자청구 끊김 공포** — "다음 달 청구 못 하면 매출 0원"
3. **직원 재교육 비용** — "선생님들 다 새로 가르쳐야"
4. **약정 기간** — "케어플 약정 6개월 남음"

→ 기능 비교표로는 절대 못 이김. **공포 4종을 하나씩 해체**해야 함.

## 03 · 이관 락인 전략 — 4단계

### Phase 1 · "데이터 공포" 해체 (2주, 즉시 착수)
- **엑셀 직임포트 완성도 끌어올리기**
  - 케어플 5종 엑셀 100% 매칭 검증 (현재 일일서비스 선생님 매칭 이슈처럼 엣지케이스 잡기)
  - 임포트 후 "케어플 vs AIHPRO 데이터 일치율" 자동 리포트 (회기 수/매출/이용자 수 비교)
  - 100% 일치 시 "이관 완료 인증서" PDF (대표 사인용)
- **"되돌리기" 안전망**
  - 임포트 결과를 케어플 원본 포맷으로 역수출 (.xlsx) — 언제든 돌아갈 수 있다는 안심
- **마이그레이션 컨시어지**
  - 무료 60일 중 첫 7일 무료 1:1 줌 이관 지원 (담당자 배정, 화면 공유)
  - 이관 완료 체크리스트 자동 발송

### Phase 2 · "청구 공포" 해체 (4-6주)
- **전자바우처 자동 청구 1차 완성**
  - 현재 .xlsx 생성 → 사회서비스원 API 직결 (가능한 지역부터)
  - 청구 실패율/반려 사유 자동 분석 대시보드
- **이중 청구 보험 모드**
  - 첫 2개월간 케어플 + AIHPRO 동시 청구 가능 (Read 모드로 케어플 데이터 sync)
  - 양쪽 결과 비교 → 100% 일치 확인 후 케어플 해지 유도
- **"청구 안 되면 환불 + 케어플 1개월치 보전"** 보증 (마케팅 카피)

### Phase 3 · "재교육 공포" 해체 (병행)
- **케어플 UI 미러 모드** (선택형)
  - "케어플처럼 보기" 토글 — 5060 원장님 익숙한 테이블 뷰
  - 점진적으로 모던 뷰로 유도
- **AI 입력 비서**
  - 음성 입력으로 일지 작성 (모바일 우선)
  - 케어플에선 PC 키보드 5분 → AIHPRO 모바일 음성 30초
  - 이게 진짜 무기 — 한 번 써보면 못 돌아감
- **부모 앱과 연동 시연**
  - 회기 끝나는 순간 부모에게 자동 리포트 발송 → 부모 만족도 ↑ → 원장이 자랑할 거리

### Phase 4 · "약정 공포" 해체 + 락인 (지속)
- **케어플 약정 잔여기간 무료 연장**
  - "케어플 약정 6개월 남았어요" → AIHPRO 60일 무료 + 잔여 6개월도 무료
  - 약정 만료일에 자동 결제 전환
- **AIHPRO만의 락인 자산 (케어플로 못 돌아가게)**
  - 스토어프론트 매출 (`/center/:slug` 도서·교구 외부 결제) — 케어플엔 없는 수익원
  - 부모 리포트 누적 히스토리 + 종단 분석 — 옮기면 부모와의 신뢰 자산 손실
  - AI 코칭 히스토리 → 이게 곧 센터의 "포트폴리오"
  - 보조금 신청 자동화 (Phase 2 이후) — 한 번 받으면 다음 분기에 또 필요

## 04 · 가격 전략

| 단계 | 가격 | 목적 |
|---|---|---|
| 0-60일 | 무료 | 데이터 락인 (이관 비용 0) |
| 케어플 약정 잔여 | 무료 | 약정 핑계 차단 |
| 정상 | 케어플 대비 30-50% | "더 좋은데 더 싸다" |
| 스토어프론트 매출 | 수수료 X (당분간) | 센터가 수익 보면 못 떠남 |

## 05 · 마케팅 메시지 3개

1. **"케어플 엑셀 5개만 올리면 끝. 60일 무료, 카드 없음."** (이관 진입 카피)
2. **"치료사가 PC 앞에 갈 시간을, 아이 앞에 쓰세요."** (모바일/AI 차별화)
3. **"부모님이 먼저 묻습니다 — '저희 센터도 이거 쓰나요?'"** (부모 앱 락인)

## 06 · 우선순위 (다음 2주 액션)

1. **케어플 엑셀 임포트 일치율 리포트 화면** — Phase 1의 핵심 무기
2. **마이그레이션 컨시어지 신청 폼** — 영업 리드 수집
3. **데이터 역수출 (.xlsx)** — "안심 보험" 표시
4. **이관 사례 1호 센터 확보** → 영상 인터뷰 → 랜딩 상단 노출
