### 1) Q3. 아이디어는 누구의 어떤 문제를 해결해주나요? (1000자, 차별점 위주)

**4단락 구조 / 공백 포함 약 950자**

1. **타깃 1 — 2030 부모 (마음 + 발달 동시 고민)**
   - ChatGPT는 임상 신뢰 부족, 병원·상담은 비용·낙인·예약대기 장벽.
   - "내 아이가 또래 대비 어디쯤인지" 객관 기준이 없는 부모의 막막함.
2. **타깃 2 — 2040 직장인 / 번아웃·불안 세대**
   - 우울·불안·번아웃 자가 점검은 받아도 다음 액션이 없음.
   - 회사에 알리기 싫고, 병원은 부담스러운 사각지대 인구 급증.
3. **타깃 3 — 기업 HR / 학교·기관 (B2B)**
   - 임직원 EAP·학생 정서지원이 형식적이고 데이터 부재.
   - 익명성을 지키면서도 부서·학급 단위 집계 리포트 필요.
4. **AIHPRO만의 차별점 (3가지)**
   - ① 7대 데이터 교차 분석 + RCI(신뢰변화지수) 임상통계 → 단순 챗봇·자가검사와 격이 다름
   - ② AI×검증 전문가 하이브리드 → 위기 신호 감지 시 외부 핫라인 대신 내부 안전망으로 즉시 연결
   - ③ 비의료 '발달 코칭 & 의사결정 보조' 카테고리 선점 → 의료법 리스크 0

### 2) Q4. 아이디어를 어떻게 실현하고 싶으신가요? (1000자, 추진 계획 + 멘토 조언)

**4단락 구조 / 공백 포함 약 970자**

1. **이미 검증된 것 (현재 상태)**
   - 7대 데이터 분석 엔진, 30일 마음 트랙(/mind-track), 전문가 매칭, B2B Job Coach 모듈 가동 중
   - 50+ 제휴기관 · 검증 전문가 네트워크 · 자동 갱신 구독 · 리워드 시스템 구축 완료
2. **모두의 창업 기간 추진 계획 (3단계)**
   - Phase 1 (1~2개월): 2030 부모·직장인 페이드 캠페인으로 PMF 정량 검증 (RCI 변화량·구독 유지율 KPI)
   - Phase 2 (3~4개월): B2B Job Coach 파일럿 3~5곳 (기업 EAP·학교 위클래스·복지기관)
   - Phase 3 (5~6개월): 임상 자문위원회 구성 + 한국형 마음건강 SaaS 표준 백서 발간
3. **멘토에게 듣고 싶은 조언**
   - 비의료 코칭 카테고리에서 의료광고법·개인정보법 라인을 더 안전하게 잡는 법
   - B2B 영업 사이클 단축 (HR·교육청·복지부 사업 연계 노하우)
   - 글로벌 진출 시 Calm·Wysa와 차별화할 KR 오리진 포지셔닝 전략
4. **장기 비전**
   - 한국형 Calm+Wysa 표준 → 일본·동남아·중동 (정신건강 인프라 부족·낙인 큰 시장) 진출.
