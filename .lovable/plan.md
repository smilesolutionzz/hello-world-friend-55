## 모두의 창업(modoo.or.kr) 신청서 작성 플랜

신청서 화면(Q1 한 줄 아이디어, Q2 배경 이야기 + 사진 첨부)에 바로 복사·붙여넣기 할 수 있는 **완성 원고**와, Q2에 첨부할 **대표 이미지 1장**을 만들어드립니다. (Q3 이후가 필요하면 동일 방식으로 이어서 작성 가능)

---

### 1) Q1. 나의 아이디어를 한 줄 소개 (100자 이내, 최소 10자)

3가지 톤을 후보로 작성하고, 그 중 메인안 1개를 권장합니다.

- **메인안 (감성 + 차별화)**  
  "AI는 정확하게, 전문가는 따뜻하게 — 마음과 발달을 30일 만에 바꾸는 비의료 코칭 플랫폼 AIHPRO"
- **대안 A (시장·기술 강조)**  
  "ChatGPT가 못 푸는 마음·발달 고민, 7대 데이터와 임상통계(RCI)로 푸는 한국형 Calm+Wysa, AIHPRO"
- **대안 B (B2B 확장 강조)**  
  "개인 마음관리부터 기업 임직원 멘탈케어까지, AI×전문가 하이브리드 코칭 SaaS 'AIHPRO'"

→ 글자 수는 모두 100자 이내로 맞추고, 메모리상의 **금지어(의료 진단/치료, MBTI, PHQ-9, 특정 가격 숫자)** 일절 미포함.

---

### 2) Q2. 아이디어를 떠올린 배경 이야기 (1000자, 도전자 경험 톤)

다음 4단락 구조로 약 900~980자 분량의 본문을 작성합니다 (도전자 1인칭 톤, 이모지 없음, 비의료 코칭 브랜딩 준수):

1. **문제 인식 (Why now)**  
   - 팬데믹 이후 한국의 우울·불안·번아웃 급증, 2030 부모의 발달 고민 폭증.  
   - 기존 ChatGPT/일반 챗봇은 임상적 신뢰가 없고, 병원·상담은 비용·낙인·접근성 장벽이 큼.
2. **개인적 계기 (도전자 스토리)**  
   - 가족·지인의 마음·발달 고민을 가까이서 보며 "AI의 정확함 × 전문가의 따뜻함"이 결합된 도구의 부재를 절감.
3. **솔루션 — AIHPRO 소개**  
   - 7대 데이터(검사·관찰일지·게임상담·음성상담·외부기록·온보딩·리포트) 교차 분석 기반 **AI 발달 코칭 리포트**.  
   - **30일 마음 트랙(/mind-track)**: Noom 스타일 일일 미션 + RCI(신뢰변화지수) 종단 추적.  
   - **전문가 매칭(/expert-hiring)**: 위기 감지 시 즉시 검증된 전문가로 연결 (외부 핫라인 대신 내부 안전망).  
   - **B2B Job Coach**: 기업 임직원 익명 코칭 + 부서 단위 집계 리포트로 확장.
4. **시장성·차별점·기대 효과**  
   - Calm + Wysa 하이브리드 포지셔닝, 의료법 리스크 없는 '발달 코칭 & 의사결정 보조' 카테고리 선점.  
   - 이미 50개 제휴기관·검증 전문가 네트워크 + 자동 갱신 구독 + 리워드 시스템으로 **유닛 이코노믹스 검증 완료**.  
   - 모두의 창업을 통해 사업화 검증 → 국내 마음건강 SaaS 표준으로 확장.

→ 1000자 카운터에 맞게 **공백 포함 980자 이내**로 정밀 컷.  
→ 표·마크다운 미사용(메모리 규칙), 자연스러운 단락 구분만 사용.  
→ "01, 02..." 식 전문 넘버링은 신청서 톤과 안 맞아 제외, 대신 단락 첫문장에 핵심 키워드 강조.

### 3) Q2 첨부용 대표 이미지 1장 생성

**사양**
- 형식: PNG, 가로형 1600×900 (모두의 창업 본문에서 잘 보이는 비율)
- 스타일: AIHPRO 브랜드 가이드 준수 — **프리미엄 화이트 미니멀리즘**, 골드 액센트(#C8B88A), 이모지·그라데이션 과용 금지
- 내용 구성 (인포그래픽 1장):
  - 좌측: "AI × 전문가" 휴먼터치 철학 카피
  - 중앙: 7대 데이터 → 통합 리포트 → 30일 마음 트랙 → 전문가 연결 흐름도
  - 우측: 핵심 지표(예: 7대 데이터 교차분석 / RCI 임상통계 / 50+ 제휴기관) — **구체 가격·매출 숫자는 미기재**(메모리 규칙)
  - 하단: AIHPRO 로고 워드마크 + "발달 코칭 & 의사결정 보조 도구" 디스클레이머 (소형)
- 생성 방식: ai-gateway 스킬의 `lovable_ai.py --image` (Gemini image preview)로 생성 → `/mnt/documents/aihpro_modoo_q2.png` 저장 → `<lov-artifact>` 태그로 다운로드 제공

> 이미지 생성 후 **자체 QA**: 이미지를 다시 열어 텍스트 잘림·오타·색상·로고 가독성 점검, 문제 시 1회 재생성.
