## 🔨 개편 1: 결제 페이지 `/token-subscription`

### 현재 문제
- "30일 마음 변화 트랙" 기능 나열만 있음 → "내가 19,900원 내면 뭘 얻는지" 안 보임
- 매일 미션 / 주간 리포트 / 전문가 상담이 **추상적인 텍스트**로만 존재

### 새 구조 (위→아래 스크롤 순서)

**섹션 1. 상단 히어로 — "당신의 30일 후 모습"**
- 헤드라인: "30일 후, 당신의 마음은 이렇게 달라집니다"
- 6개 고민 칩 (우울/불안/수면/자존감/관계/육아) — 사용자가 자기 고민을 인식
- 가격: ₩19,900 (₩39,800 → 50% 할인) + "일시불 · 자동결제 없음" 강조

**섹션 2. Before / After 비교 카드 (NEW)**
- 좌: "지금의 나" (어두운 톤) — "잠을 못 자고 / 아침이 두렵고 / 작은 일에 무너져요"
- 우: "30일 후의 나" (밝은 톤) — "수면 루틴이 잡히고 / 아침 5분 마음 정리 습관 / 무너져도 다시 일어서는 회복탄력성"
- 화살표 가운데 "₩19,900 · 30일 프로그램"

**섹션 3. "30일간 매일 무슨 일이 일어나나요?" — 데일리 미션 미리보기 (NEW)**
- Day 1 / Day 7 / Day 15 / Day 30 샘플 미션 카드 4개
- 각 카드: "오전 8시 도착 · 5분 코칭 · 실제 메일 미리보기 (썸네일)"
- 예: "Day 1 — 오늘의 마음 점수 측정하기 / Day 7 — 첫 주 변화 리포트 도착 / Day 30 — 변화 종합 PDF"

**섹션 4. 주간 리포트 미리보기 (NEW)**
- 실제 리포트 화면 모형 1장 (블러 처리 + "결제 후 매주 일요일 발송" 라벨)
- "RCI 신뢰변화지수 기반 · 박사급 분석" 신뢰 뱃지

**섹션 5. 전문가 상담권 1회 = 어떻게 쓰는지 (NEW)**
- 3단계 미니 플로우: ① 상담권 활성화 → ② AI가 내 데이터 기반 전문가 매칭 → ③ 40분 1:1 상담 예약
- "필요할 때만 쓰면 됩니다 (강제 아님)" 안심 카피

**섹션 6. 7일 환불 보장 + FAQ + 최종 CTA**
- 환불 정책 큰 카드
- FAQ 3개: "자동 결제 되나요? / 30일 끝나면? / 환불은 어떻게?"
- 고정 CTA: "₩19,900 결제하고 내일 아침 8시부터 시작하기"

## 🔨 개편 2: 전문가 페이지 `/expert-hiring`

### 현재 문제
- 전문가 카드 그리드만 나옴 → "전문가가 나한테 뭘 어떻게 해주는지" 안 보임
- 상담 방식(채팅/영상/대면), 진행 흐름, 상담 후 무엇을 받는지 불분명

### 새 구조 (페이지 최상단에 추가, 기존 전문가 그리드는 유지)

**섹션 0 (NEW · 페이지 최상단). "전문가 상담은 이렇게 진행됩니다" 5단계 플로우**
1. **고민 선택** — 우울/불안/수면/자존감/관계/육아 중 선택
2. **AI 매칭** — 당신의 고민에 가장 잘 맞는 전문가 3명을 자동 추천 (기존 `match-consultation-expert` 활용)
3. **시간 예약** — 캘린더에서 40분 슬롯 선택
4. **1:1 상담** — 화상 또는 채팅 (방식 명시)
5. **후속 케어** — 상담 요약 리포트 + 30일 마음 트랙 연계

각 단계는 아이콘 + 한 줄 설명 + 소요 시간 표시.

**섹션 0.5 (NEW). "어떤 전문가를 만나나요?" 신뢰 카드 3개**
- 자격 검증: "국가공인 자격증 · 5년 이상 경력만 입점"
- 실제 후기: "평균 4.8/5점 · 누적 1,200건 상담"
- 비밀 보장: "상담 내용 100% 비공개 · 회사·가족 통보 절대 없음"

**섹션 0.6 (NEW). 가격 명세 카드**
- 40분 1:1 상담: ₩49,000
- 구독자 할인: 30일 트랙 결제자는 1회 무료 (이미 있는 로직 명시)
- "추가 상담 필요시 30% 할인" 등 명시

**기존 섹션 (유지)**
- 카테고리 필터 + 전문가 카드 그리드
- 협력기관 탭

## 🧪 검증 후 확인할 것

개편 후 사용자가 결제 페이지에 들어가면 **3초 안에** 다음 3가지 질문에 답할 수 있어야 함:
1. "이게 뭐야?" → 30일 마음 코칭 프로그램
2. "내가 뭘 얻어?" → Before/After + 데일리 미션 미리보기로 직관적으로 보임
3. "전문가는 뭐야?" → 5단계 플로우 + 신뢰 카드로 명확

전문가 페이지도 동일하게 **3초 룰** 적용.
