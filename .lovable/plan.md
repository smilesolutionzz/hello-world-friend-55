## 제안: 30일 마음 트랙을 상단 네비 단독 메뉴로 승격

### 왜 좋은 판단인가
- **단일 핵심 결제 상품(₩19,900)**이자 PMF 중심축인데, 지금은 `검사·리포트` 드롭다운 4번째 항목으로 묻혀 있음 (`UnifiedNavigation.tsx` L74~85)
- 사용자가 "이 회사가 파는 게 뭔지"를 한눈에 인지하지 못함 → 전환 손실
- Calm·Wysa 같은 경쟁사도 핵심 프로그램은 항상 최상위 단독 노출

### 변경 사항 (UnifiedNavigation.tsx)

**Desktop (lg+) — 데스크톱 상단바**
- `홈` 버튼 바로 옆, 다른 드롭다운들보다 **앞**에 단독 버튼으로 배치
- 시각적으로 차별화된 강조 스타일 (gold 액센트 `#C8B88A` 또는 primary solid + 살짝 빛나는 테두리)
- 상태별 라벨:
  - 미구독: `30일 마음 트랙` + `₩19,900` 작은 서브 라벨 또는 `NEW` 뱃지
  - 결제 완료(needs_baseline): `30일 마음 트랙` + `시작하기` 뱃지
  - 진행 중(active): `30일 마음 트랙` + `Day N/30` 뱃지
- 클릭 시 `/mind-track`으로 이동 (기존과 동일)
- `검사·리포트` 드롭다운에서는 **제거** (중복 방지)

**Mobile 햄버거 메뉴**
- 메뉴 최상단(홈 바로 아래)에 동일하게 단독 항목으로 분리
- 같은 강조 스타일 + 상태별 뱃지 적용
- `검사·리포트` 섹션에서는 제거

**MobileBottomTab은 손대지 않음**
- 이미 `구독` 탭(Crown 아이콘)이 `/token-subscription`으로 가는데, 이건 별도 검토 필요. 이번엔 상단 네비에만 집중.

### 디자인 토큰 (메모리 준수)

- "AI"/"진단" 표현 금지, "30일 마음 트랙"만 사용
- 가격은 메모리에 적힌 대로 하드코딩 금지 → `MIND_TRACK_PRICE` 상수 import해서 표시
