# 케어플 대비 차별화 실행 플랜

기관장 피드백을 두 갈래로 풀어냅니다.
1) **가독성**: 케어플 주간표(노란 헤더·꽉 찬 색 블록·텍스트 잘림)보다 한눈에 잡히는 차분한 캘린더로 리디자인.
2) **전환 이유**: 단순 일정관리 SaaS가 아니라 "AIHPRO에만 있는 것"을 가시화 — 비교 랜딩 + 60일 무료 + 1클릭 마이그레이션 약속.

---

## ACT 1 — 주간 캘린더 가독성 리디자인 (가장 큰 페인포인트)

대상 파일: `src/pages/b2b-center/console/SchedulePage.tsx` (현 1,497 LOC)

### 핵심 원칙 (케어플 약점 → 우리의 강점)

```text
케어플 현 상태                      AIHPRO 리디자인
─────────────────────────────────  ─────────────────────────────
노란 풀-사츄레이션 헤더             흰 헤더 + 얇은 골드 라인(#C8B88A)
이벤트마다 진한 채도 배경           좌측 4px 컬러 바 + 흰 카드
[이름] 프로그램 텍스트 잘림         이름 1줄 + 프로그램 1줄 ellipsis,
                                    hover 시 호버카드로 풀 정보
8색 이상 혼재                       치료영역 5색 팔레트로 통일
요일 헤더 = 작은 회색 텍스트        요일 헤더 = 큰 숫자+요일, 오늘 강조
일정 미니맵 사이드바 빽빽           기본 접힘, 필터/검색 우선 노출
```

### 구체 작업

1. **이벤트 카드 컴포넌트 신설** — `src/components/b2b-center/schedule/EventCard.tsx`
   - 좌측 4px 컬러 스트라이프, 흰 배경, 시간 11px / 이름 13px semibold / 프로그램 12px muted
   - 30분 슬롯이면 1줄 압축 모드, 60분+이면 2줄 모드
   - hover 시 `HoverCard`로 이용자·치료사·메모·결제상태 풀 표시
2. **컬러 토큰 정리** — `src/lib/scheduleColors.ts`
   - 8+색 → 5색 (재활/상담/평가/그룹/기타) 의미 단위로 축소
   - 모두 채도 35% 이하, 명도 88%+의 파스텔로 통일
3. **주간 그리드 리팩토링** — `WeekGridView` 추출
   - 헤더: `2026.06.15 (월) · 오늘 8건` 형식, sticky
   - 종일/오전/오후/저녁 4구간 zebra 행
   - 충돌 이벤트는 가로 분할(50/50) + tooltip
4. **밀도 토글 추가** — Compact / Comfortable / Spacious (localStorage)
   - 기관장 데모에선 Comfortable 기본, 치료사는 Compact 선호 가정
5. **상단 정보 막대** — "이번 주: 예약 142 · 완료 38 · 취소 4 · 가용시간 87%"

### 기술 메모
- 기존 일정 데이터 스키마 변경 없음 (`schedule_events` 그대로)
- shadcn `HoverCard`, `ToggleGroup` 활용, 신규 라이브러리 없음
- 모바일에선 자동으로 "일" 뷰로 폴백
