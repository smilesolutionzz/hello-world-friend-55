# MVP 론칭을 위한 개선 계획

## 1단계: 핵심 기능 단순화 (1-2주)

### A. 메인 플로우 단순화
**현재**: 9가지 분산된 기능
**개선**: 핵심 3가지만 유지
1. 간단 관찰일지 (무료 체험)
2. AI 분석 리포트 (유료)
3. 전문가 상담 연결 (프리미엄)

### B. 사용자 여정 최적화
```
입장 → 간단 설문 → 무료 관찰 체험 → AI 분석 맛보기 → 구독 유도 → 전문가 연결
```

## 2단계: 신뢰성 확보 (1주)

### A. 인증 시스템 강화
- ✅ 이미 구현: AuthenticationGuard
- ✅ 이미 구현: 개선된 에러 메시지
- 추가 필요: 자동 재로그인 시스템

### B. 데이터 무손실 보장
- 로컬 스토리지 백업
- 오프라인 작성 지원
- 자동 저장 기능

### C. 결제 시스템 안정화
- 결제 실패 시 재시도 로직
- 환불 정책 명시
- 이용약관/개인정보처리방침

## 3단계: 핵심 가치 집중 (1주)

### A. 타겟 고객 명확화
**1차 타겟**: 자녀 발달이 걱정되는 부모
**2차 타겟**: 자신의 심리상태가 궁금한 성인

### B. 가치 제안 단순화
"24시간 AI 심리전문가와 함께하는 우리 가족 마음 건강 관리"

### C. 온보딩 최적화
- 30초 내 핵심 기능 체험 가능
- 회원가입 없이 1회 무료 체험
- 결과에 놀란 사용자의 자연스러운 구독 유도

## 4단계: 비즈니스 모델 단순화

### A. 요금제 단순화
**현재**: 토큰제 + 구독제 복잡한 구조
**개선**: 
- 무료: 월 1회 관찰일지
- 베이직(월 9,900원): 무제한 관찰 + AI 분석
- 프리미엄(월 29,900원): 베이직 + 전문가 상담 1회

### B. 수익화 우선순위
1. 구독 수익 (월 경상 수익)
2. 전문가 매칭 수수료 (확장성)
3. 기업용 솔루션 (스케일링)

## 론칭 준비도 체크리스트

### 필수 (차단 요소)
- [ ] 인증 오류 완전 해결
- [ ] 핵심 플로우 3분 내 완주 가능
- [ ] 결제 시스템 안정성 테스트
- [ ] 개인정보처리방침/이용약관 정비

### 권장 (경쟁력)
- [ ] 모바일 최적화
- [ ] 소셜 로그인 연동
- [ ] 카카오톡 공유 기능
- [ ] 초기 사용자 온보딩 가이드

### 마케팅 준비
- [ ] 랜딩페이지 최적화
- [ ] 무료 체험 결과 공유 기능
- [ ] 추천인 리워드 시스템
- [ ] 초기 전문가 풀 확보 (최소 20명)

## 예상 론칭 일정
- **개선 작업**: 3-4주
- **베타 테스트**: 1-2주 (지인/지인가족 대상)
- **소프트 론칭**: 2주 (제한된 마케팅)
- **정식 론칭**: 이후

## 성공 지표 (MVP)
- **가입 전환율**: 무료 체험 → 가입 15% 이상
- **결제 전환율**: 가입 → 유료 전환 5% 이상  
- **재방문율**: 7일 내 재방문 30% 이상
- **NPS 점수**: 40점 이상 (추천 의향)
