## 서비스 소개

### AIHPRO란?
AIHPRO(AI하이라이트프로)는 **"데이터로 읽는 마음"**을 모토로 하는 AI 기반 심리건강 플랫폼입니다.

### 주요 특징
- 🧠 **AI 기반 심리분석**: 최신 AI 기술을 활용한 정확한 심리상태 분석
- 📊 **데이터 기반 인사이트**: 객관적인 데이터를 통한 맞춤형 솔루션 제공
- 👨‍⚕️ **전문가 연결**: 자격을 갖춘 심리전문가와의 직접 상담 기회
- 📱 **24시간 접근**: 언제 어디서나 이용 가능한 온라인 서비스
- 👪 **가족 통합 관리**: 개인부터 가족 전체까지 통합 관리

## 회원가입 및 로그인

### 사용자 유형 선택
회원가입 시 다음 중 하나를 선택하세요:

#### 1. 개인 사용자
- **대상**: 나와 가족을 위한 심리검사를 원하는 일반인
- **혜택**: 개인 맞춤형 심리검사, AI 상담, 관찰일지
- **온보딩**: 개인 니즈 기반 맞춤 추천

#### 2. 전문가
- **대상**: 심리상담사, 의료진, 교육 전문가
- **혜택**: 전문가용 도구, 환자/내담자 관리, 전문 리포트
- **온보딩**: 전문 분야 및 자격 정보 입력

#### 3. 기관 관계자
- **대상**: 학교, 병원, 상담센터 등 기관 소속
- **혜택**: 대량 검사, 기관용 대시보드, 통계 분석
- **온보딩**: 기관 정보 및 활용 목적 설정

### 회원가입 절차
1. **사용자 유형 선택**
2. **기본 정보 입력**
   - 이름, 이메일, 비밀번호
   - 전화번호 (선택)
   - 생년월일 (선택)
3. **이메일 인증** (설정에 따라)
4. **유형별 온보딩 진행**

## 사용자 유형별 기능

### 개인 사용자 기능

#### 맞춤형 니즈 평가
- **관심 영역 선택**: 우울감, 불안, 스트레스, ADHD, 아동발달 등
- **대상 설정**: 본인, 자녀, 가족 전체
- **경험 수준**: 초보자, 중급자, 고급자
- **맞춤 추천**: 선택에 따른 최적 검사 패키지 제안

#### 이용 가능한 검사
- **3분 심리검사**: 빠른 기본 상태 체크 (2토큰)
- **우울증 선별검사**: 전문적인 우울증 진단 도구 (2토큰)
- **불안장애 검사**: 불안 증상 정도 측정 (2토큰)
- **ADHD 자가진단**: 주의력결핍 과잉행동장애 검사 (3토큰)
- **언어발달 검사**: 아동 언어 발달 평가 (3토큰)
- **프리미엄 종합검사**: 전문가급 상세 분석 (8토큰)

### 전문가 기능

#### 전문가 프로필 설정
- **전문 분야**: 임상심리사, 상담심리사, 아동심리전문가 등
- **자격 정보**: 자격증, 경력 기간, 근무지
- **상담 대상**: 연령대별, 영역별 전문성
- **프로필 완성**: 자기소개, 상담 가능 시간

#### 전문가용 도구
- **고급 분석 도구**: 더 정교한 심리분석 알고리즘
- **환자 관리**: 내담자별 진행 상황 추적
- **전문 리포트**: 임상용 상세 분석 보고서
- **이미지 생성**: 치료용 시각 자료 생성

### 기관 관계자 기능

#### 기관 프로필 설정
- **기관 유형**: 교육기관, 의료기관, 상담센터, 공공기관 등
- **담당 역할**: 상담사, 심리사, 의료진, 교사 등
- **서비스 대상**: 아동, 청소년, 성인, 가족, 직원 등
- **활용 목적**: 선별검사, 심리평가, 상담, 교육, 연구 등

#### 기관용 도구
- **대량 검사**: 다수 대상자 동시 검사
- **통계 대시보드**: 기관 전체 현황 분석
- **그룹 관리**: 부서별, 학급별 그룹 관리
- **리포트 생성**: 기관용 종합 보고서

## 심리검사 가이드

### 이용 가능한 검사 목록

#### 기본 심리검사
1. **3분 심리검사** (2토큰)
   - 빠른 기본 상태 체크
   - 전반적인 심리상태 평가
   - 추가 검사 추천

2. **우울증 선별검사** (2토큰)
   - 전문적인 우울증 진단 도구
   - AI 기반 상세 분석 제공
   - 맞춤형 관리 방안 제시

3. **불안장애 검사** (2토큰)
   - 불안 증상 정도 측정
   - 다양한 불안 유형 분석
   - 대처 전략 제공

4. **ADHD 자가진단** (3토큰)
   - 주의력결핍 과잉행동장애 검사
   - 연령별 맞춤 평가
   - 관찰일지 연계 추천

5. **언어발달 검사** (3토큰)
   - 아동 언어 발달 평가
   - 연령별 발달 단계 비교
   - 발달 촉진 방법 제시

6. **스트레스 검사** (2토큰)
   - 스트레스 수준 측정
   - 스트레스 원인 분석
   - 관리 전략 제공

7. **방어기제 분석** (3토큰)
   - 심리적 방어기제 유형 파악
   - AI 기반 상세 해석
   - 건강한 대처 방법 제안

8. **6차원 성격진단 (HEXACO)** (3토큰)
   - 정직성-겸손성, 정서성, 외향성 등 6가지 차원 분석
   - AI 기반 종합 성격 분석
   - 개인 성장 방향 제시

9. **프리미엄 종합검사** (8토큰)
   - 전문가급 상세 분석
   - 다차원적 심리평가
   - 맞춤형 개선 방안

### 검사 선택 가이드

#### 처음 이용하는 경우
1. **3분 심리검사**로 시작 (2토큰)
2. 결과에 따라 추가 검사 추천 받기
3. **AI 상담사**로 결과 해석 (무료)

#### 특정 증상이 있는 경우
- **우울감**: 우울증 선별검사 → AI 상담사
- **불안증상**: 불안장애 검사 → AI 상담사
- **스트레스**: 스트레스 검사 → 관찰일지
- **집중력 문제**: ADHD 자가진단 → 관찰일지
- **아동 발달**: 언어발달 검사 → 관찰일지
- **성격 이해**: 6차원 성격진단 → AI 상담사
- **심리 패턴**: 방어기제 분석 → AI 상담사

#### 종합적인 분석을 원하는 경우
- **프리미엄 종합검사** (8토큰)
- 전문가급 상세 분석
- 맞춤형 개선 방안 제시

### 검사 진행 방법

#### 사전 준비
1. **조용한 환경** 확보
2. **충분한 시간** 확보 (검사별 소요시간 확인)
3. **집중할 수 있는 상태** 확인

#### 검사 중 주의사항
- **솔직하게** 답변하기
- **첫 번째 느낌**으로 선택하기
- **너무 오래 고민하지** 않기
- **중간에 중단하지** 않기

#### 결과 확인 및 활용
1. **즉시 결과 확인** 가능
2. **AI 분석 리포트** 자동 생성
3. **개선 방안** 맞춤 제시
4. **PDF 다운로드** 가능
5. **전문가 상담** 추천 (필요시)

### 검사 결과 관리

#### PDF 파일 저장 및 공유
- **PDF 다운로드**: 모든 검사 결과를 PDF 파일로 저장 가능
- **파일명 형식**: `검사명_이름_날짜.pdf` (예: 우울증검사_홍길동_20250116.pdf)
- **공유 기능**: 
  - 이메일로 전송
  - 카카오톡/SNS 공유
  - 제휴기관에 제출
  - 전문가에게 전달

#### 제휴기관 시스템 연동
- **자동 업데이트**: 검사 완료 시 제휴기관에 자동 전송
- **실시간 공유**: 제휴기관 담당자가 즉시 확인 가능
- **양방향 소통**: 
  - 기관 → 플랫폼: 검사 요청, 일정 관리
  - 플랫폼 → 기관: 결과 전송, 진행 상황 공유
- **데이터 보안**: 암호화된 안전한 전송

#### 제휴기관과의 협업
1. **기관 등록**: 기관 코드 입력으로 연결
2. **검사 의뢰**: 기관에서 필요한 검사 추천
3. **결과 공유**: 
   - 사용자 동의 하에 자동 공유
   - PDF 형식으로 전문자료 제공
   - Zoom 회의, 이메일 등 다양한 방법 지원
4. **후속 관리**: 
   - 기관의 피드백 수신
   - 추가 검사 제안
   - 지속적인 모니터링

#### 검사 기록 관리
- **내 검사 기록**: 모든 검사 이력 조회
- **비교 분석**: 이전 결과와 비교
- **진행 상황**: 시간별 변화 추적
- **공유 이력**: 언제 누구와 공유했는지 확인

## 관찰일지 작성법

### 관찰일지의 목적
- **객관적 행동 기록**
- **패턴 파악**
- **변화 추적**
- **전문가 상담 자료**

### 효과적인 작성법

#### 1. 행동 기록
```
시간: 오후 3시
상황: 숙제하는 시간
행동: 책상에 앉았다가 5분마다 일어나서 돌아다님
지속시간: 30분 중 실제 집중시간 10분
```

#### 2. 감정 상태
```
아침: 평온함 (5점/10점)
점심: 약간 짜증남 (3점/10점)
저녁: 매우 피곤함 (2점/10점)
```

#### 3. 특이사항
```
- 친구와 싸운 후 말수가 줄어듦
- 좋아하는 음식도 잘 안 먹음
- 평소보다 일찍 잠자리에 들어감
```

### 작성 주기
- **매일**: 기본 상태 기록
- **주간**: 주요 변화 요약
- **월간**: 전체적인 패턴 분석

## 전문가 서비스

### 전문가 매칭 시스템

#### 매칭 기준
- **지역**: 거주지 기반
- **전문분야**: 증상/연령별
- **언어**: 한국어/기타

#### 상담 방식
- **온라인 화상상담**

#### Q: 검사 결과가 정확한가요?

#### Q: 전문가 상담은 어떻게 예약하나요?
